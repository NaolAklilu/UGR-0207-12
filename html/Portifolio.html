Mini Projects

Analog clock made by Python

Analog clock is a device or a tool which is used to read the time of the day. It has three hands. The Shortest hand indicates the Hours, the longer hand indicates the Minutes and the longest hand indicates the Seconds. This project made a digital analog clock that resembles the physical analog clock. Analog clock is one of my projects I have done before.

This analog clock is made using a turtle module in python. It used to tell both the time, the date, month and year. It's simple to use and modify. The code of this project uses Turtle to establish a drawing context and instantiates a Turtle object in a variable called drawing_pen. The name drawing_pen is a giveaway that this is the object that will draw to the screen.

2048 game using python

2048 is a challenging number puzzle for kids and grown ups. The game's objective is to slide numbered tiles on a grid to combine them to create a tile with the number 2048. However, one can continue to play the game after reaching the goal, creating tiles with larger numbers.

According to neurologist Judy Willis, 2048 may look like a series of doubling number numbers but really it's a dopamine goldmine. And dopamine, which is a very useful neurotransmitter, is also kind of like an addictive drug the body produces naturally. It boosts both pleasure and perseverance, while decreasing stress.

how to play: Use your arrow keys to move the tiles. Tiles with the same number merge into one when they touch. Add them up to reach 2048!
